# 老年人健康管理系统 - AI改进计划提示词

## 项目背景
这是一个基于HarmonyOS开发的老年人健康管理应用，采用ArkTS语言和ArkUI框架。项目目标是提供简洁、易用的健康管理工具，适合老年人日常使用。

## 当前项目状态分析

### 已实现功能
1. **健康打卡模块** - 基础的健康状态记录和历史查看
2. **用药提醒模块** - 药品信息管理和服药状态跟踪
3. **健康指标模块** - 血压、血糖、心率数据记录
4. **紧急联系模块** - SOS功能和联系人管理
5. **数据持久化** - 基于Preferences API的本地存储

### 技术架构
- **前端框架**: ArkUI + ArkTS
- **状态管理**: @State装饰器
- **数据存储**: StorageService单例模式
- **UI设计**: 底部导航 + 页面切换
